# Assets — Logo Fragmentos

A top bar da home usa o logo oficial em `fragmentos-logo.png`.

```
docs/assets/fragmentos-logo.png
```

Notas:
- O logo é apresentado a `height:50px` (largura automática), sobre a barra branca.
- Se um dia tiveres o logo em **SVG**, coloca `fragmentos-logo.svg` aqui e muda a
  extensão em `<img class="nav-logo" src="assets/fragmentos-logo.png">` no index.html
  (o SVG escala sem perder nitidez em ecrãs de alta densidade).
- Evita espaços no nome do ficheiro (usa hífens).
- Só o index (home) usa este logo. As top bars dentro dos dashboards não foram alteradas.
